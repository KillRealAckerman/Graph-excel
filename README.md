# Графики по файлам формата .xlsx
Мне показалось, что с графиками в Excel не очень удобно работать, поэтому, используя Chart.JS и SheetJS, я написал код, который строит диаграммы по загруженным файлам. Каждая строка таблицы - отдельная линия. Сейчас это выглядит приметивно, НО просто. В дальнейшем планитруется добавить возможность открыть график на всю страницу для таблиц с большим количеством данных.
